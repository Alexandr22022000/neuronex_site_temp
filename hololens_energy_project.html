Hololens Instructions Project

Приложение для Microsoft Hololens с графическим представлением инструкций для энергетической и горнодобывающей промышленностей. Приложение позволяет увеличивать скорость и качество обучения персонала промышленных предприятий работе со сложным оборудованием. Большое количество оборудования в горнодобывающей и энергетической промышленности представляют серьезную опасность для жизни и здоровья человека, внедрение подобных технологий приводит к снижению количества ошибок при работе с подобным оборудованием и помогает сохранить жизнь и здоровье персонала.

В примере на видео изображена инструкция по работе с КСР-6-400 (большой удлинитель для карьеров, в нем соединяются 2 кабеля с напряжение 6000 вольт). Приложение разработано совместно с центром компьютерного инжиниринга Кемеровского Государственного Университета.

Платформа: Universal Windows Platform (Hololens)
Часы разработки: 240 часов
Область применения: Энергетическая и горнодобывающая промышленность
Назначение системы: Обучение персонала работе со сложным и опасным оборудованием
